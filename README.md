## ‚öôÔ∏è Setup Steps

### Backend Setup

1. Navigate to the backend folder
```bash
cd backend
Install backend dependencies

npm install
```

Create a .env file inside the backend folder and add:
```bash
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/lead_management
JWT_SECRET=your_secret_key
```

Start the backend server
```bash
npm run dev
```

Backend runs on:
```bash
http://localhost:5000
```
Frontend Setup

Navigate to the frontend folder
```bash
cd frontend
```

Install frontend dependencies
```bash
npm install
```

Start the frontend application
```bash
npm start
```

Frontend runs on:
```bash
http://localhost:3000
```
